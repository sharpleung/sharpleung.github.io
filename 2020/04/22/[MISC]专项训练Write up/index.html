[MRCTF2020]Hello_ misc

binwalk分离出来一个加密的压缩包。爆破无果。

用StegSolve图片中分离出一张图片得到密码：

解压出来是TTL.用脚本转换为字符

import re

txt = open("out.txt",'r')
line = txt.readlines()
number = []
flag = ""
for i in line:
    number.append(int(i))
for i in number:
    if(i == 63):
        flag += "00"
    elif(i == 127):
        flag += "01"
    elif(i == 191):
        flag += "10"
    else:
        flag += "11"
strr = re.findall(r'.{8}', flag)
for i in strr:
    print(chr(int(i,2)),end="")
